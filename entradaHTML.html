HTML

Las paginas HTML son documentos de texto que usan principalmente elementos para definir la estructura de la pagina.

Para definir los elementos se usan etiquetas y van entre parentesis angulares < >

El contenido de un elemento se encierra entre etiquetas y existen etiquetas de apertura < / b > y de cierre < / b >

Las etiquetas pueden tener atributos los cuales se especifican en la etiqueta de inicio. Los atributos contienen informacion adicional acerca del contenido
