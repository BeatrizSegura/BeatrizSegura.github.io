Estructura Semantica de HTML

Etiquetas

Para escribir una pagina en HTML necesitamos conocer las principales etiquetas para darle formato

A continuacion explicare algunas de las etiquetas:

  • Encabezados(heading): Se representan con la etiqueta h1, h2, h3, h4, h5 y h6 siendo h1 el de mayor importancia.
  • Parrafos: Se representan con la etiqueta p
  • Saltos de linea: Se representan con la etiqueta br, estos son muy utiles ya que al estar escribiendo una pagina en HTML y deseamos agregar espacio entre el texto no funciona hacerlo con Enter, se tiene que agregar esta etiqueta.
  • Espacios en blanco: Para agregarlos se usa un caracter especial llamado "Non-breaking space" y es usado para agregar mayor espacio entre las palabras o unir 2 palabras que no se pueden separar a pesar del tama√±o de la ventana.
  • Negritas: Se representa con la etiqueta b que es una abreviatura de bold.
  • Cursivas: Se representan con la etiqueta i que es una abreviatura de italic.
  • Resaltado: Se representa con la etiqueta strong y visualmente tiene el mismo efecto que el bold pero sirve para darle cierta importancia al texto al que se le aplique.
  • Enfasis: Se representa con la etiqueta em y como la etiqueta anterior sirve para darle un significado o importacia al texto.
  • Caracteres especiales: Esto no es propiamente una etiqueta si no un conjunto de caracteres o numeros que desplegaran en la pantalla algun caracter que tal vez en HTML sea reservado o que ni siquiera existe en nuestro teclado como es el caso de ©, para desplegarlos se utilizara el simbolo & seguido del nombre del caracter o numero correspondiente. Para ver la lista de estos caracteres puedes accesar a w3schools para la lista de entidades de HTML.

Estructura Semantica

Ahora que conocemos las etiquetas "necesarias" para poder empezar a redactar una pagina HTML, es necesario tambien conocer la estructura semantica de la paginas para poder redactarla como se debe

Consta principalmente de 8 apartados principales para que la pagina este correctamente estructurada

  • a)Encabezado: Indicado con la etiqueta header
  • b)Navegacion: Indicado con la etiqueta nav
  • c)Contenido principal: Indicado con la etiqueta main
  • d)Seccion: Indicado con la etiqueta section
  • e)Articulo: Indicado con la etiqueta article
  • f)Barra lateral: Indicado con la etiqueta aside
  • g)Pie de pagina: Indicado con la etiqueta footer
  • h)Divisiones: Indicado con la etiqueta div
