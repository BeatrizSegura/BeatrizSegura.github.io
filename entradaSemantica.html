Estructura Semantica de HTML

Para escribir una pagina en HTML necesitamos conocer las principales etiquetas para darle formato

A continuacion explicare algunas de las etiquetas:

  • Encabezados(heading): Se representan con la etiqueta h1, h2, h3, h4, h5 y h6 siendo h1 el de mayor importancia.
  • Parrafos: Se representan con la etiqueta p
  • Saltos de linea: Se representan con la etiqueta br, estos son muy utiles ya que al estar escribiendo una pagina en HTML y deseamos agregar espacio entre el texto no funciona hacerlo con Enter, se tiene que agregar esta etiqueta.
  • Espacios en blanco: Para agregarlos se usa un caracter especial llamado "Non-breaking space" y es usado para agregar mayor espacio entre las palabras o unir 2 palabras que no se pueden separar a pesar del tama√±o de la ventana.
  • Negritas: Se representa con la etiqueta b que es una abreviatura de bold.
  • Cursivas: Se representan con la etiqueta i que es una abreviatura de italic.
