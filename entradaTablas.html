Elementos de una tabla

Las tablas son muy utilizadas al crear una pagina en HTML ya que regularmente se usan para desplegar informacion de una forma mucho mas entendible para el usuario

Sus etiquetas principales son:

  • table: Es la etiqueta principal, dentro de la cual van las demas etiquetas para construir la tabla.
  • caption: Esta etiqueta sirve para darle una descripcion a nuestra tabla, sobre el contenido o a que esta relacionada.
  • thead: Esta etiqueta representa el encabezado de la tabla, dentro de ella llevara otras etiquetas para los elementos que conforman la tabla.
  • tfoot: Con esta etiqueta se agrega un pie de "pagina" a la tabla.
  • tbody: Esta etiqueta sirve para agregar los elementos del cuerpo de la tabla.
  • tr: Con esta etiqueta se agregan filas a la tabla.
  • th: Con esta etiqueta se agregan celdas al encabezado de la tabla.
  • td: Con esta etiqueta se agregan celdas al cuerpo de la tabla.
