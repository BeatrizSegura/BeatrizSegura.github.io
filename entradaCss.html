CSS

Este lenguaje nos permite cambiar la apariencia de los elementos HTML, ya sea dandole color, fuentes diferentes, etc. Como todo lenguaje lleva una sintaxis general conformada por el selector { declaraciones } .
Las declaraciones se conforman de propiedad : valor; .


Tipos de Estilos

Ya como su nombre lo dice esta hoja nos servira para definir el estilo de la pagina HTML y existen principalmente 5:

  • Por default: Este estilo lo da el navegador a cualquier pagina HTML.
  • Del usuario: Este estilo se agrega en el archivo local, donde se tiene el codigo en HTML.
  • Hoja de estilo externa: En este estilo se utiliza la etiqueta link para hacer referencia la archivo que contiene las declaraciones. Esta es la forma mas recomendada.
  • Estilo interno: Utiliza la etiqueta style dentro del mismo archivo HTML.
  • Estilo en linea: Se asigna la propiedad y el valor como atributo del elemento.

Selectores CSS

Como se explico anteriormente los selectores son los que nos ayudan a definir los estilos de una pagina y para hacer esto existen diferentes tipos de selectores y aqui explicara los 5 principales:

  • Selector de tipo: Selecciona un tipo de elemento HTML y aplica estilo a todos los elementos del mismo tipo, por ejemplo si le queremos aplicar a todos los parrafos una alineacion seria: p { text- align: justify; }
  • Selector descendente: Aplica el estilo a elementos que estan contenidos dentro de otros elementos por ejemplo: p b { color: red; }
  • Selector hijo: Similar al descendente pero mas restrictivo.
    Aplica el estilo solamente a los elementos que sean hijos directos de otro elemento p > b { color: blue; }
  • Selector por ID: ID es un atributo de un elemento HTML y se utiliza para aplicar un estilo de manera particular. Este selector utiliza el simbolo #. #introduccion { color: blue; }
  • Selector por clase: Aplica estilo a un conjunto de elementos, utiliza el simbolo .
    .especial { color: red; }
  • Selector universal: Aplica el mismo estilo a todos los elementos de la pagina *{ color: red; }

Los estilos se heredan, si aplicas un estilo a un elemento, todos los elementos dentro de el tendran el mismo estilo, pero se pueden modificar por medio de declaraciones en cascada
