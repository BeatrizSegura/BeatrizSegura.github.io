CSS

Este lenguaje nos permite cambiar la apariencia de los elementos HTML, ya sea dandole color, fuentes diferentes, etc.

Como todo lenguaje lleva una sintaxis general conformada por el selector { declaraciones } .
Las declaraciones se conforman de propiedad : valor; .


Tipos de Estilos

Ya como su nombre lo dice esta hoja nos servira para definir el estilo de la pagina HTML y existen principalmente 5:

  • Por default: Este estilo lo da el navegador a cualquier pagina HTML.
  • Del usuario: Este estilo se agrega en el archivo local, donde se tiene el codigo en HTML.
  • Hoja de estilo externa: En este estilo se utiliza la etiqueta link para hacer referencia la archivo que contiene las declaraciones. Esta es la forma mas recomendada.
  • Estilo interno: Utiliza la etiqueta style dentro del mismo archivo HTML.
  • Estilo en linea: Se asigna la propiedad y el valor como atributo del elemento.
