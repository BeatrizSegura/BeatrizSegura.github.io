HTML

El significado de las siglas de HTML es: HiperText Markup Language (Lenguaje de marcado para hipertexto) y fue creado por Tim Berners Lee aproximadamente en el año 1990 en el CERN para compartir informacion con los cientificos

Estructura Semantica de un documento HTML

Una pagina en HTML tiene que llevar una estructura especifica, mas alla de lo visual sirve para la accesibilidad de la pagina

Enlaces en un documento HTML

Al poner enlaces en HTML pueden exitir diferentes opciones para hacerlo, ya que pueden ser: relativos, absolutos o contenido para descargar

Tablas en HTML

Las tablas pueden ser muy utiles al momento de desplegar informacion en una pagina y estas llevan una estructura especial

CSS

El significado de las siglas CSS es: Cascading Style Sheets (Hojas de estilo en cascada) y fue creada aproximadamente en el año 1996
