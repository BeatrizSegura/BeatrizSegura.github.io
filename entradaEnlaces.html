Enlaces en HTML

Como se puede leer en la introduccion existen 3 tipos principalmente de enlaces en HTML. Los cuales estan explicados a continuacion:

  • Relativo: Esto significa que hace el enlace en la misma pagina HTML en la que nos encontramos
  • Absoluto: Esto significa que hace el enlace hacia otra pagina en la red
  • Contenido para descargar: Como su nombre lo dice, el enlace no abrira una pagina si no que descargara algun contenido

Para poder usar los enlaces en nuestras paginas HTML se usa la etiqueta a que significa anchor Esta etiqueta no lleva etiqueta de cierre y puede llevar hasta 3 atributos en ella:

  • href: lleva entre comillas la direccion o archivo al que se va a hacer el enlace href= "enlace" si aun no se tiene definido el enlace se puede usar el caracter # para hacer un enlace nulo
  • target: lleva entre comillas _blank para que al hacer click sobre el enlace abra una ventana de navegacion nueva o _self para abrirlo en la ventana actual y este lo toma por default
  • download: este atributo no lleva nada entre las comillas y puede ir o no dependiendo del enlace

Imagenes

Aunque las imagenes no son propiamente un enlace tambien son muy usadas en las paginas HTML y como los enlaces no lleva etiqueta de cierre. Algunos de sus atributos pueden ser:

  • src: lleva entre comillas el nombre de la imagen con extension src ="nombre.png"
  • alt: lleva entre comillas un texto alternativo por si la imagen no carga o para el aspecto de accesibilidad alt= "texto alternativo"
