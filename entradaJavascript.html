JavaScript

Como cualquier otro lenguaje orientado a objetos JavaScript usa objetos, propiedades, eventos y metodos en su estructura.

Propiamente en el lenguaje puede llegara a usar:

  • Operadores Aritmeticos: Como + - * / %
  • Operadores Racionales: Como == != < >
  • Operadores Logicos: Como && || !
  • Operadores de asignacion: Como = += -= *=
  • Condicionales: Como if, if-else, switch
  • Ciclos: Como while, do-while, for, break

Los principales tipos de datos que se utilizan son:

  • Cadena
  • Numero, que pueden ser enteros o decimales
  • Booleano
  • Arreglos
  • Funciones

Una diferencia significativa a otro lenguajes es que al terminar una sentencia o metodo el ; es Opcional

Para implementar el codigo de JavaScript a un documento HTML hay 3 opciones:

  • Interno: Dentro del head o el body del documento HTML
  • Externo: En un archivo separado y se enlaza con el atributo src y es el mas recomendado
  • En linea: Se agregaria como un atributo en el documento HTML
